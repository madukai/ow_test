Closures -- A closure is a feature in JavaScript where an inner function or nested function and it has access to the outer (enclosing) function’s variables — a scope chain.

Hoisting -- All variable declarations using "var" are hoisted/lifted to the top of their functional/local scope (if declared inside a function) or to the top of their global scope (if declared outside of a function) regardless of where the actual declaration has been made.

Promise -- Is an object that may produce a single value some time in the future. The result can be a successful (returns a code e.g 200) or fail (returns a code e.g. 400).

XMLHttpRequest -- Is used to make a http request to a server or an API, to get the necessary data need.
